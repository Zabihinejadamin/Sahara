# Sahara Raiders Assets

This folder contains game assets and graphics. All files are currently placeholders.

## Required Assets (to be added later):

### Icons & UI
- `icon.png` - App icon (512x512)
- `camel.png` - Spy/camel icon for scouting
- `caravan.png` - Caravan sprite
- `oasis.png` - Oasis/water source sprite
- `ruins.png` - Ancient ruins sprite

### Tiles & Textures
- `desert_hex.png` - Desert terrain hex tile
- `dunes_hex.png` - Dunes terrain hex tile
- `sandstorm_overlay.png` - Sandstorm effect overlay

### UI Elements
- `button_normal.png` - Normal button texture
- `button_pressed.png` - Pressed button texture
- `progress_bar.png` - Custom progress bar texture

## Asset Sources:
- Kenney.nl Desert Pack
- itch.io Hex Tiles
- OpenGameArt.org Desert Assets
- Custom SVG/PNG creations

## Notes:
- All sprites should be designed for mobile (high DPI)
- Hex tiles should be 64x64 or 128x128 pixels
- Use PNG format with transparency where needed
